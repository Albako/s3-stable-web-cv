Home Server

Prosty domowy serwer stworzony przy użyciu laptopa Asus 556LD i Switcha keepLINK 2,5Gbps w wersji zarządzalnej. Serwer posiada statyczne prywatne IP oraz OS serwera to Debian bez GUI.
